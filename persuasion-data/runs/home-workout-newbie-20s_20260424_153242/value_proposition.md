# Mofit — 폰 하나로 운동이 교정되는 AI 트레이너

## 1. 가치제안 요약 (랜딩페이지 카피)

### 메인 카피

**"폰 세워두고 손바닥 1초. 그게 전부입니다."**

iPhone 카메라만으로 스쿼트를 자동으로 세고, 자세를 실시간 교정합니다. 별도 장비도, 회원가입도 필요 없습니다. 앱 켠지 20초 안에 첫 스쿼트가 카운트됩니다.

### 서브 카피

- **자동 카운팅**: 폼이 맞을 때만 1회로 인정. 수기 체크 필요 없음.
- **개인화 AI 코칭**: 내 키·몸무게·목표와 최근 7일 운동 데이터 기반 피드백. 일반론 아닙니다.
- **오프라인 우선**: 로그인 없이 바로 시작. 데이터는 내 폰에만. (AI 코칭만 로그인 필요)
- **100% 무료 (MVP 단계)**: 광고 없음. 카운팅·기록은 영원히 무료.

### 타깃 시나리오

> "헬스장 3개월 등록했는데 2주 못 갔어요. PT는 비싸고, 유튜브 보고 따라해도 내 폼이 맞는지 모르겠어요."

Mofit은 바로 이 지점을 겨냥합니다. 폰을 벽에 기대 세워두고, 손바닥 1초를 카메라에 보여주면 5초 카운트다운 후 추적이 시작됩니다. 스쿼트가 끝날 때까지 폰에 손대지 않아도 됩니다.

## 2. 구체적 기능 설명 (미팅 / 데모 전달 시)

### 2.1 온보딩 (최초 1회, 약 30초)

성별 → 키 → 몸무게 → 체형(마른/보통/근육질/통통한) → 목표(체중 감량/근력 증가/체형 개선). 회원가입·이메일 요구 없음. 입력은 전부 폰 로컬(SwiftData)에 저장.

### 2.2 홈탭

- 오늘의 기록 요약 (세트, rep, 시간)
- "운동 시작" 형광초록 버튼
- 운동 종류 선택 (스쿼트 / 푸쉬업 / 런지 / 플랭크 — MVP에서는 내부적으로 전부 스쿼트로 처리, 향후 확장)

### 2.3 트래킹 화면 (카메라)

상태 머신:

```
대기 → 손바닥 1초 → 카운트다운 5초 → 운동 추적
     → 손바닥 1초 → 세트 완료 → 카운트다운 5초 → 다음 세트 → ... → 종료 버튼
```

- **스쿼트 판정**: Apple Vision Framework의 `VNDetectHumanBodyPoseRequest` 로 hip-knee-ankle 각도 분석. 각도 > 160°일 때 서있음, < 100°일 때 앉음. 서있음 → 앉음 → 서있음 = 1 rep.
- **손바닥 판정**: `VNDetectHumanHandPoseRequest` 로 5개 손가락이 1초 펴진 상태 감지 → 세트 시작/종료 신호.
- 관절 포인트가 화면에 오버레이로 표시되어 자세 교정 가능.
- 프리뷰 30fps, 분석 15fps 샘플링 (배터리 절약 + 스쿼트 속도에 충분).

### 2.4 기록탭

- 상단: 최근 7일 가로 스크롤 날짜바
- 선택 날짜의 세션 리스트 (운동 종류, 세트수, 총 rep, 시간, 시작 시각)
- 스와이프 삭제

### 2.5 AI 코칭탭 (로그인 필요)

- 하루 2회: 운동 전 동기부여 / 운동 후 피드백 (자정 리셋)
- Claude Sonnet 4.6 모델 사용, 서버에서 프록시
- 전달 컨텍스트: 사용자 프로필 + 최근 7일 요약(운동일수, 총세션, 총rep, 일평균rep) + 일별 추이(rep/세트/세트당 평균)
- API 실패 시 횟수 차감 안 함

### 2.6 프라이버시 / 데이터

- **비로그인**: 모든 데이터 로컬(SwiftData). 외부 전송 0.
- **로그인 시**: 프로필/운동기록/AI 피드백이 서버(Supabase, JWT 인증)에 저장. Claude API도 서버 경유.
- 카메라 영상은 어디에도 저장·전송되지 않음. 실시간 분석 후 폐기.

## 3. 가격 / 플랜

### 단계 0 (현재, MVP)

**완전 무료. 광고 없음. 회원가입 없음.** 기본 기능(카운팅·기록·자세 오버레이)은 영구 무료로 유지할 계획.

### 단계 1 (가설, 수개월 후)

- 기본 카운팅·기록·자세 오버레이: 영구 무료
- 프리미엄 (가설): AI 코칭 호출 횟수 상향 + 주간 리포트 + 운동별 폼 점수 상세

단계 1 BM은 사용자 수·리텐션 검증 후 확정 예정. 지금 설치하는 사용자는 "완전 무료 MVP 단계"를 확약받는 것과 같음.

### 단계 2 (가설, 장기)

B2B — 헬스장/PT 숍용 고객 운동 데이터 대시보드. 개인 사용자에게는 영향 없음.

## 5. 리스크 (솔직한 고지)

- **스쿼트 외 운동**은 UI는 있으나 MVP에서는 내부적으로 스쿼트 판정 로직을 재사용. 푸쉬업·런지·플랭크의 정확한 판정은 단계적으로 추가 예정.
- **rep 카운트 정확도**: Apple Vision 기반이므로 조명·각도·옷차림에 따라 오차 가능. 폰은 발끝에서 약 2m, 허리 높이에 세워두는 것이 권장 환경.
- **iOS 17+ 전용** (iPhone XS 이상). Android·iPad 미지원.

## 6. 즉시 시작하는 방법

1. App Store에서 "Mofit" 검색 (출시 예정)
2. 앱 실행 → 온보딩 30초 → 홈 → 운동 시작
3. 폰을 벽에 기대 세우기 → 화면 안에 서서 손바닥을 1초 펴기
4. 5초 후 카운트 시작. 끝나면 손바닥 1초 더 → 세트 완료.

설정도, 로그인도, 결제도 없습니다. 카메라 권한만 허용하면 끝.
