- 파일: `01_keyman_initial.md` — "ADR-017", "ADR-013" 같은 내부 문서 식별자를 키맨이 자연스럽게 인용. `personality_notes`는 "앱 스토어 리뷰와 인스타 리일 추천으로 앱 선택"하는 일반 소비자로 설정돼 있는데, ADR 번호까지 언급하는 것은 소비자 관점에서 부자연스러움. 가치제안 본문에 해당 ID가 노출된 결과로 보이나, **페르소나가 반응 시 "소비자 언어"로 번역해 표현하도록** 시뮬 프롬프트 보정 후보.
