# 사용자 흐름 (User Flow)

## 1. 최초 실행
```
앱 설치 → 열기 → 온보딩
  성별 선택 (자동 진행)
  → 키 입력 (다음 버튼, 검증)
  → 몸무게 입력 (다음 버튼, 검증)
  → 체형 선택 (자동 진행)
  → 목표 선택 (자동 진행, 온보딩 완료)
  → 홈탭
```
- 각 단계에서 뒤로가기로 이전 단계 수정 가능
- 경고 범위 값: Alert "정말 {값}이 맞나요?" → 확인/취소
- 거부 범위 값: 다음 버튼 비활성화 + 빨간 안내 텍스트

## 2. 운동 흐름 (핵심)
```
홈탭
  → 운동 종류 탭 → 바텀시트 그리드 → 선택 → 시트 닫힘
  → "운동 시작" 탭
  → (최초 1회) 카메라 권한 요청
  → 트래킹 화면 진입

트래킹 상태 머신:
  [대기] "손바닥을 보여주세요"
    → 손바닥 1초 인식
  [카운트다운] "5, 4, 3, 2, 1" (중앙 크게)
    → 카운트 완료
  [운동 추적] 관절 오버레이 + rep 카운트
    → 손바닥 1초 인식
  [세트 전환] "세트 N 완료!" 잠깐 표시 → 바로 카운트다운 5초
    → 카운트 완료 → [운동 추적] (다음 세트)

  언제든 → 종료 버튼(빨간 원) 탭
    → 기록 저장 → 홈 이동 → 폭죽 효과
```

## 3. 기록 열람
```
기록탭 → 날짜바에서 날짜 선택 (좌 스와이프로 이전)
  → 해당 날짜 세션 리스트
  → 세션 스와이프 → 삭제
```

## 4. AI 코칭
```
AI코칭탭
  → "운동 전 피드백" 또는 "운동 후 피드백" 탭
  → 로딩 → AI 피드백 카드 표시 (최신 카드 펼쳐짐)
  → 사용한 버튼 비활성화
  → 과거 카드 탭 → 펼침/접힘
```

## 5. 프로필 수정
```
홈탭 → 우상단 프로필 버튼
  → 풀스크린 모달 (5개 값 수정)
  → 저장 → 모달 닫힘
```

## 6. 재방문
```
앱 열기 → onboardingCompleted 체크
  → true: 홈탭
  → false: 온보딩
```
